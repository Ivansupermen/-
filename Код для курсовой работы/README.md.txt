Калькулятор стоимости заемного капитала.

Программа для расчета стоимости заемного капитала с учетом налогового щита и дополнительных расходов.

Описание:
Я разработал эту программу в рамках курсовой работы по финансовому менеджменту. Она позволяет рассчитать реальную стоимость заемных средств с учетом налоговых вычетов и дополнительных затрат.

Возможности:
1.Расчет номинальной стоимости капитала
2.Учет налогового щита
3.Расчет эффективной годовой ставки
4.Автоматическое тестирование
5/Валидация входных данных

Формулы расчета:
1.Номинальная стоимость: (проценты/cумма)*100%
2.После налога: номинальная*(1-налоговая ставка)
3.Эффективная ставка: ((1+номинальная/100)^1-1)*100%+(расходы/сумма)*100%
